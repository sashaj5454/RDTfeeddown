Installation
============

To install RDTfeeddown, run:

.. code-block:: bash

   pip install rdtfeeddown

Or, to install from source:

.. code-block:: bash

   git clone https://github.com/sashaj5454/rdtfeeddown.git
   cd rdtfeeddown
   pip install .

Dependencies
------------

- Python >= 3.11
- numpy
- pyqtgraph
- matplotlib
- pytest
- tfs-pandas
- scipy
- PyQt5/6
- QtPy
