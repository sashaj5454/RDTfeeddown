Installation
============

To install RDTFeeddown, run:

.. code-block:: bash

   pip install rdtfeeddown

Or, to install from source:

.. code-block:: bash

   git clone https://github.com/yourusername/rdtfeeddown.git
   cd rdtfeeddown
   pip install .

Dependencies
------------

- Python >= 3.8
- numpy
- pyqtgraph
- matplotlib
- pytest
- tfs
- QtPy

Run the GUI using:

.. code-block:: bash

   python -m rdtfeeddown
